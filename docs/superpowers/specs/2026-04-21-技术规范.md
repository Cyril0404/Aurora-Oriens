## 一、电商方案：Plan A · WooCommerce

**方案选择：** 开源自托管电商（不用Shopify）

| 项目 | 选择 |
|------|------|
| **电商平台** | WooCommerce（WordPress插件） |
| **服务器** | 腾讯云轻量服务器（$5-10/月）或 Vultr / DigitalOcean |
| **域名** | Namecheap / 阿里云 |
| **支付** | Stripe / PayPal（通过WooCommerce插件） |
| **邮件** | Klaviyo / Mailchimp |

### 为什么选 WooCommerce
- 全球使用量最多（40%+电商网站）
- 开源免费，插件生态成熟
- 完整订单管理/购物车/库存/优惠券
- 可接入 Stripe / PayPal / 支付宝（部分插件）
- 我们已有静态HTML原型 → 可迁移设计风格到WooCommerce主题

## 三、设计规范（WooCommerce 主题需遵循）

### 色彩（不可修改）
```css
--color-ivory:      #FFFFF0   /* 主背景，90%使用 */
--color-ink:        #1A1A1A   /* 主文字/按钮 */
--color-gold:       #C9A84C   /* 强调（价格/hover） */
--color-red-subtle: #B85C5A   /* 点缀（badge/错误） */
```

### 字体
| 用途 | 字体 |
|------|------|
| 标题/品牌 | Playfair Display, Georgia, serif |
| 正文/按钮 | Inter, -apple-system, sans-serif |

### 审美原则（不可偏离）
- ✅ Heritage + Everyday Luxury + Artisan + Intentional
- ❌ 过度奢华/批量生产感/土豪风/过时传统中国风
- 文案：高端旅行杂志风格，故事驱动
- 禁用词：cheap/fast fashion/bargain（用honest pricing）
