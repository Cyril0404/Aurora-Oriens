## 三、购物车系统

```javascript
// localStorage key: 'ot_cart'
// 数据格式:
[
  {
    id: "de-001",        // 商品ID，与products.json对应
    name: "Jade Bangle Bracelet",
    price: 45,            // 美元
    image: "https://...", // 产品图URL
    quantity: 2           // 数量
  }
]

// 核心函数（见开发手册）
getCart()           // 从localStorage读取
saveCart(items)     // 写入localStorage
cartTotal(items)   // 计算总价（美元）
cartCount(items)    // 计算总数量
updateBadge()       // 更新导航栏角标（带pop动画）
openCart() / closeCart()
openCheckout() / closeCheckout()
updateSteps()       // 更新结算步骤指示器
renderCartDrawer()  // 渲染购物车面板
validateAddress()   // 表单验证
```

## 七、SEO技术要求

- 所有页面 `<title>` 格式：`页面名 - Oriental Treasure`
- `<meta name="description">` 每页独立，150字符以内
- OG tags：og:title / og:description / og:image（每页独立）
- 图片：必须加 `alt` 属性
- robots.txt：`Disallow: /src/`（静态文件不索引）
- sitemap.xml：包含所有可访问页面URL
